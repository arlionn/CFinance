## 关于这本书

这本在线电子书是我讲授《公司金融》课程时积累的笔记和代码示例的结集，旨在帮助读者理解公司金融中的核心概念，并通过 Python 代码实现相关的计算和分析。书中的内容涵盖了公司金融的基本理论、财务报表分析、资本预算、风险管理等多个方面。

## 如何使用本书？

这本书是使用 VScode 编辑器 + Quarto 插件编写，进而通过 Github pages 发布的。详情参见 [Quarto books](https://lianxhcn.github.io/quarto_book/)。 

-   你可以 [Fork](https://github.com/arlionn/CFinance/fork) 本书的 [github 仓库](https://github.com/arlionn/CFinance)，然后你的电脑上演练书中的例子。
-   你需要做一些基本的配置，参见 [安装和环境配置](https://lianxhcn.github.io/research_with_AI/body/01_install_Python_Anaconda.html)
-   书稿中还有不少错漏之处，还请不吝指出：[arlionn\@163.com](mailto:arlionn@163.com){.email}。
